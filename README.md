# Apache Web Server Project

##  Description
This project demonstrates hosting a static website using Apache on AWS EC2.

##  Features
- Installed and configured Apache2
- Hosted a static HTML website
- Configured security groups (port 80)
- Tested using curl and browser

## Tech Stack
- Linux (Ubuntu)
- AWS EC2
- Apache2
- Git & GitHub

##  How to Access
http://35.154.154.211

##  Project Setup
1. Install Apache
2. Place files in /var/www/html
3. Start Apache service
4. Access via browser
